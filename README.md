# Task Checklist

Task Checklist is a small Next.js app for managing daily tasks in a clean, focused interface. Users can add tasks, mark them as completed, and remove them when they are no longer needed.

The task list is stored in the browser with local storage, so items remain available after a page refresh without requiring a backend or database.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks from the list
- Persist tasks in local storage
- Responsive layout with a styled background image

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).
